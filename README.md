# Getir Case Study


## Install
*NOTE:* This API built with Node 15

Clone the repo
```sh
git clone https://github.com/teathedev/getir-case-study.git
cd getir-case-study
```

Install packages
```sh
npm i
```

Create the .env file
```sh
cp sample.env .env
```

You can configure .env file as you want

## Start API
For development
```sh
npm start-dev
```

For production
```sh
npm start
```

## Test

Start test
```sh
npm test
```
